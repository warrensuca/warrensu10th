Flexbox Study Guide

Flexbox works with 1d rows and columns. Use

"display: flex;"
to begin using! When creating container to use Flexbox, keep in mind you are working on the layer that contains the elements, not the individual elements. The individual elements are called flex items, and are simply accessed with
.items
or
.items:nth-of-type.

Flex-Direction

Flex-Direction tells Flexbox if your content is a row or column. The attributes you set for your container will change accordingly. For a row (default), the main axis would be left to right and the cross axis would be up to down.

Justify Content/Align Items

Justify Content works on the main axis and can space out or position items. Align items does the same except it works on the cross axis. A common use is

justify-content: space-evenly;

Items

Specific items have 3 main attributes. Shorthand the attributes with

flex: grow, shrink, basis
