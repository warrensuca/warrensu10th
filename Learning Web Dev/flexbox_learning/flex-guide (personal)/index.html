Flexbox works with 1d rows and columns. Use "display: flex;" to begin using! When creating container to use Flexbox, keep in mind you are working on the layer that contains the elements, not the individual elements. The individual elements are called flex items, and are simply accessed with .items or .items:nth-of-type.

flex-direction tells flexbox if your div is a row or column. The attributes you set for your container will change accordingly.
