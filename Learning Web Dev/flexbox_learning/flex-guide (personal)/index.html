Flexbox Study Guide

Flexbox works with 1d rows and columns. Use "display: flex;" to begin using! When creating container to use Flexbox, keep in mind you are working on the layer that contains the elements, not the individual elements. The individual elements are called flex items, and are simply accessed with .items or .items:nth-of-type.

Flex-Direction tells Flexbox if your div is a row or column. The attributes you set for your container will change accordingly. For a row (default), the main axis would be left to right and the cross axis would be up to down.

  • flex-direction: row is the default
  • flex:direction: column flips the main and cross axis

Justify Content works on the main axis and can

  • Space out items
  • Position group of items

Align Items works on the cross axis and can

  • Space out items
  • Position group of items
  • Stretch items

Specific items have 3 main attributes. Shorthand the attributes with "flex: grow, shrink, basis"

  • flex-grow determines how much a flex item should grow to fit availabe space. If given flex-grow: 2, the item will take up 2x the space as default
  • flex-shrink is used very rarely and determines how much a flex item will shrink relative to other items when there isn't enough space
  • flex-basis is the intial size of the item relative to the other items
